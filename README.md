# 基于ESP-IDF的四旋翼飞控
  一款可以简单部署的ESP32的飞控，可直接用于四旋翼无人机，且包含有完善的ESP-IDF的外设二次封装库和常用硬件库(C++)  

# 目前已完善的功能
  1.所需的ESP-IDF的外设库的C++二次封装  
  2.常用硬件库(MPU9250,General ESC(PWM))  
  3.串级PID  
  4.基于互补滤波的姿态估计  

# 说明
  1.EnderFCS为飞控代码的工程文件夹，请配置好ESP-IDF的开发环境后编译此工程(这里推荐VSCode的ESP-IDF插件)  
  2.resourses为一些资料
